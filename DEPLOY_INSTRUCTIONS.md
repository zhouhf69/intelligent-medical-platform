# 🚀 智能病历平台 - 部署指南

## ⚡ 快速部署（30秒完成）

### 方法：使用Netlify Drop

#### 步骤1：访问Netlify
打开浏览器访问：
```
https://app.netlify.com/drop
```

#### 步骤2：拖拽部署
1. 打开文件资源管理器
2. 找到 `D:\养老市场\public` 文件夹
3. 将文件夹拖拽到Netlify页面的指定区域
4. 等待几秒钟

#### 步骤3：获取网站地址
- Netlify会自动生成一个随机网址
- 例如：`https://xxx-xxx-xxx.netlify.app`
- 点击网址即可访问

---

## 📦 备选方案：使用deploy.zip

如果拖拽文件夹不方便，可以使用zip文件：

1. 访问 https://app.netlify.com/drop
2. 将 `D:\养老市场\deploy.zip` 文件拖拽到指定区域
3. 等待部署完成

---

## 🔧 配置自定义域名（可选）

部署成功后，可以在Netlify设置中配置自定义域名：

1. 访问站点管理页面
2. 点击 "Domain settings"
3. 点击 "Add custom domain"
4. 输入您的域名
